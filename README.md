# Book

A `ledger` cli wrapper that allows me to write extensions easily.

## Getting started

		cd
		git clone https://github.com/silasb/book.git .book

For bash users:

		echo 'eval "$($HOME/.book/bin/book init -)"' >> ~/.bash_profile
		exec bash

For zsh users:

		echo 'eval "$($HOME/.book/bin/book init -)"' >> ~/.zshenv
		source ~/.zshenv
